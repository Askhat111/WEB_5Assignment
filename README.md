Asphalt-8 Coffeehouse Website
A responsive coffee shop website with modern features and interactive elements.

Team Members and Roles
Team: Makan
Asanali - Services & Gallery pages, animations, FAQ system.
Askhat - Blog & Contact pages, form validation, search functionality.
Asylkhan- Main page, authentication system, theme implementation.

Project Topic is a website for "Asphalt-8 Coffeehous.

Core Requirements Met:
- Responsive Design - Works on all devices
- Light/Dark Mode - Persistent theme switching
- User Authentication - Sign up, login, profile system
- Form Validation - Email, phone, password validation
- Search & Filter - Blog and FAQ search with localStorage
-Creative Animations - Typing effects, hover animations, image effects
- External API Integration - Movie search functionality

Interactive Elements:
- User registration and login system
- Contact forms with validation
- Newsletter subscription
- Star rating system for blog posts
- Image galleries with hover effects
- FAQ accordion with search
- Animated counters and typing text

External APIs Used
OMDb API- Movie database integration for entertainment section - Used on About page for
customers to search films available at the coffeehouse
